### Übung (msf Basics)

`search -h` … Hilfe zum Befehl `search`

search type:auxiliary name:ssh … genauere Spezifikationen  
`info` … zeigt genauere Informationen an  
`use <dir>` … in das Modul wechseln  
`options` … options anzeigen  
`set <option>` … Werte der options verändern  
`run` … ausführen

`setg` … set gobal setzt options für alle Module

### Übung (nmap FTP scan)

```shell
nmap --script=vuln -p 21 172.20.10.2
```

`ftp-vsftpd-backdoor:`  
`IDs:  CVE:CVE-2011-2523`

Eine shell wird ohne Authentifizierung an `Port 6200` geöffnet, wodurch Befehle ausgeführt werden können.   
intrusive bedeutet, dass aktiv versucht wird die Schwachstelle auszunutzen und nicht nur Informationen zu sammeln. Dadurch sind sie nicht bei `-sC` & `-A` dabei, um unbeabsichtigte Angriffe zu vermeiden.
